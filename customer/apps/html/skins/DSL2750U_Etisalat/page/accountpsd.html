Account Password

You can access the DSL Router through one of the three user accounts: admin, support, and user.

User "support" can access the DSL Router for maintenance and diagnosis. It is usually used by an ISP technician and cannot be used locally.

User "user" can access the DSL Router, to view configuration settings and statistics.

Use the fields below to change or create passwords. Note: Password cannot contain a space.

Helpful Hints...

In this page, you can change the password for accessing the Web management interface of the router. For security concerns, it is recommended to change the default passwords of user "admin" and user "user" immediately. Each password you set should consist of 1-16 characters. Please set a password that is easy for you to remember or write it down and keep it in a safe place for future reference. If you forget the password, you have to reset the router to factory default settings and all the configurations on the router will be lost.
